Installation

How to install dependencies and structure your app.

1

Create project

Start by creating a new project. We support all major frameworks.

npm create vite@latest my-project -- --template react-ts
2

Add Tailwind CSS

Install Tailwind CSS and its peer dependencies via npm, and then run the init command to generate both tailwind.config.js and postcss.config.js.

Terminal 
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p
3

Configure Paths

Add the paths to all of your template files in your tailwind.config.js file.

tailwind.config.js 
/** @type {import('tailwindcss').Config} */
module.exports = {
  content: [
    "./index.html",
    "./src/**/*.{js,ts,jsx,tsx}",
  ],
  theme: {
    extend: {},
  },
  plugins: [],
}
